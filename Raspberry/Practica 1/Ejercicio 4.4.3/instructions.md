# 4.4.3 Escala musical

Escribe un programa que haga sonar el altavoz con las notas Do, Mi y Sol (de
la quinta octava) durante tres segundos cada una de ellas. Las frecuencias de estas
notas son:

```
| Nota | Frecuencia |
|------|------------|
| Do   | 523 Hz     |
| Mi   | 659 Hz     |
| Sol  | 784 Hz     |
```
